# EventDesign

## Описание проекта

EventDesign - это  приложение для управления событиями, разработанное с целью помочь пользователям эффективно планировать и отслеживать важные мероприятия. 

### Основные возможности

При первом запуске пользователь попадает на экран приветствия, который предлагает два основных варианта действий: регистрация нового аккаунта или вход для существующих пользователей.

#### Ключевые функции:

1. **Управление событиями**: 
   - Создание, редактирование и удаление событий
   - Подробная информация о каждом событии (название, категория, место, дата, описание)
   - Возможность добавления в избранное

2. **Категоризация**:
   - Создание собственной системы классификации событий
   - Гибкое управление категориями

3. **Избранное**:
   - Персональная подборка важных мероприятий
   - Быстрый доступ к приоритетным событиям

4. **Настройки**:
   - Переключение между темной и светлой темами интерфейса


5. **Отчетность**:
   - Генерация детальной статистики по событиям
   - Фильтрация отчетов по периоду, категориям, пользователям
   - Экспорт в Excel и PDF
